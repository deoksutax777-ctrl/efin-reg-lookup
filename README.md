# 전자금융업 · 부가통신사업자 등록현황 조회

업체명을 검색하면 전자금융업 등록/말소 여부와 부가통신사업자 신고 여부를 한 번에 확인하는 단일 HTML 도구.

## 데이터 기준일
- 부가통신사업자 신고현황: **2026년 5월말** (과학기술정보통신부 게시자료)
- 전자금융업 등록·말소 현황: **2026-07-13** (금융위원회 게시자료)

데이터는 `index.html` 안에 스냅샷으로 내장되어 있으며, 이후 갱신분은 자동 반영되지 않는다.
원본 엑셀이 갱신되면 파일을 재생성해야 한다.

## 사용법
- 검색창에 업체명을 한 줄에 하나씩 입력 (여러 건 동시 검색 가능)
- ㈜ · (주) · 주식회사 표기 차이는 자동 정규화, 부분 일치 검색
- 결과: 전자금융업 등록내역(업종별) / 말소·취소 내역 / 부가통신 신고내역 / 종합판정 뱃지
